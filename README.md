# Loan-Approval-Prediction
Machine Learning project to predict loan approval using applicant demographics, financial information, and credit history. Includes data preprocessing, exploratory data analysis (EDA), Logistic Regression model, performance evaluation, and business insights.
# Loan Approval Prediction using Machine Learning

## Project Overview

This project predicts whether a loan application will be approved based on customer information such as income, education, marital status, loan amount, and credit history.

The objective is to automate loan approval decisions and help financial institutions reduce manual effort while improving consistency.


## Problem Statement

Banks receive thousands of loan applications every day.

Manual approval is time-consuming and may introduce bias.

This project builds a Machine Learning model that predicts loan approval using historical customer data.

## Business Insights

- Credit history is the strongest predictor of loan approval.
- Higher income applicants have better approval chances.
- Married applicants have slightly higher approval rates.
- Customers with poor credit history are mostly rejected.
